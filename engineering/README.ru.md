# Инженерия и разработка

Полный стек технической разработки — от архитектуры и бэкенда до DevOps, безопасности и мобильных приложений. 23 агента, покрывающих все инженерные дисциплины.

## Агенты

### Бэкенд-архитектор (`engineering-backend-architect.md`)

Старший бэкенд-архитектор, специализирующийся на проектировании масштабируемых систем, архитектуре баз данных, API и облачной инфраструктуре.

**Ключевые компетенции:**
- Микросервисная архитектура и модульные монолиты
- Оптимизация схем БД для масштабирования
- Проектирование RESTful и GraphQL API

**Результаты работы:** спецификации системной архитектуры, схемы БД с индексацией, паттерны API-архитектуры, стратегии декомпозиции сервисов.
