# Пространственные вычисления и XR

Данный раздел содержит коллекцию специализированных агентов, предназначенных для разработки приложений пространственных вычислений, иммерсивных XR-интерфейсов и высокопроизводительных систем 3D-рендеринга. Каждый агент обладает уникальной экспертизой -- от нативной разработки под Apple Vision Pro до кроссплатформенных браузерных XR-решений.

---

## Инженер пространственного рендеринга macOS/Metal (`macos-spatial-metal-engineer.md`)

Специалист по нативной разработке на Swift и Metal, создающий высокопроизводительные системы 3D-рендеринга и пространственных вычислений для macOS и Vision Pro. Разрабатывает иммерсивные визуализации, обеспечивающие бесшовную интеграцию между macOS и Vision Pro через Compositor Services и RemoteImmersiveSpace.

**Ключевые возможности:**
- Реализация инстансированного Metal-рендеринга для 10 000--100 000 узлов при 90 кадрах в секунду
- Создание эффективных GPU-буферов для данных графов (позиции, цвета, связи)
- Проектирование алгоритмов пространственной компоновки (силовые, иерархические, кластерные)
- Потоковая передача стереокадров на Vision Pro через Compositor Services
- Оптимизация производительности Metal: фрустум-каллинг, LOD, тройная буферизация, пулы ресурсов
- Интеграция с Vision Pro: отслеживание взгляда, распознавание жестов щипка, рейкаст-тестирование
- GPU-физика для расчёта компоновки графов с использованием вычислительных шейдеров

**Результаты работы:**
- Конвейер Metal-рендеринга с инстансированной отрисовкой узлов и рёбер
- Интеграция с Vision Pro Compositor для стереоскопического вывода
- Система пространственного взаимодействия (взгляд, жесты, выделение)
- GPU-шейдеры для силовой компоновки графов

**Ключевые возможности:**

- Оптимизация GPU-рендеринга для множественных стеклянных окон и 3D-контента
- Поддержка доступности: VoiceOver и пространственная навигация для иммерсивных интерфейсов

**Результаты работы:**
- Нативные visionOS-приложения с волюметрическими интерфейсами SwiftUI
- Реализация эффектов Liquid Glass с настраиваемыми режимами отображения
- Многооконная пространственная архитектура с управлением жизненным циклом
- Системы жестов и взаимодействия в волюметрическом пространстве

## Специалист по интеграции терминалов (`terminal-integration-specialist.md`)

Эксперт по эмуляции терминалов, оптимизации рендеринга текста и интеграции SwiftTerm в современные Swift-приложения. Создаёт надёжные и производительные терминальные окружения, которые ощущаются нативно на платформах Apple, сохраняя совместимость со стандартными терминальными протоколами.

**Ключевые возможности:**
- Полная поддержка стандартов VT100/xterm: ANSI escape-последовательности, управление курсором, состояния терминала
- Интеграция SwiftTerm в SwiftUI с корректным управлением жизненным циклом
- Оптимизация рендеринга текста через Core Graphics для плавной прокрутки и частых обновлений
- Эффективное управление буферами для больших терминальных сессий без утечек памяти
- Паттерны SSH-интеграции: мост ввода-вывода, управление состоянием соединения, обработка ошибок
- Кроссплатформенный рендеринг терминалов для iOS, macOS и visionOS
- Поддержка доступности: VoiceOver, динамический шрифт, вспомогательные технологии

**Результаты работы:**
- Встроенные терминальные компоненты на базе SwiftTerm для SwiftUI-приложений
- Оптимизированные системы рендеринга текста с управлением цветовыми схемами и шрифтами
- Интеграция SSH-потоков с эмулятором терминала
- Решения для управления множественными терминальными сессиями и окнами
