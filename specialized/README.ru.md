# Узкоспециализированные агенты

Нишевые эксперты для сложных отраслевых задач, не вписывающиеся в другие категории. Охватывают блокчейн, комплаенс, госзакупки, рекрутинг, управление знаниями, межкультурную навигацию и многое другое.

---

## Агенты

### Оператор кредиторской задолженности (`accounts-payable-agent.md`)

Автономный специалист по платёжным операциям — обработка платежей поставщикам, счетов подрядчиков и повторяющихся платежей через крипто, фиат и стейблкоин-рельсы.

**Ключевые компетенции:**
- Оптимизация маршрутизации платежей
- Верификация счетов и проверки идемпотентности
- Ведение аудит-трейлов
- Управление реестром поставщиков

**Результаты работы:** подтверждения исполнения платежей, сводки кредиторской задолженности, управление реестром поставщиков.

### Оператор графа идентичностей (`identity-graph-operator.md`)

Специалист по разрешению общих идентичностей в мульти-агентных системах. Обеспечивает детерминистическое сопоставление одной и той же сущности реального мира разными агентами.

**Ключевые компетенции:**
- Нечёткое сопоставление сущностей
- Предложения по объединению/разделению
- Обнаружение конфликтов и оценка доверия
- Аудит-трейлы решений по объединению/разделению

**Результаты работы:** канонические ID сущностей, оценки уверенности, аудит-трейлы.

**Ключевые компетенции:**
