# Тестирование и контроль качества

Данный раздел содержит набор специализированных агентов, отвечающих за тестирование, контроль качества и оценку производственной готовности программных продуктов. Каждый агент выполняет уникальную роль в процессе обеспечения качества -- от аудита доступности и анализа результатов тестирования до проверки производительности и финальной сертификации готовности к развертыванию.

---

## Аудитор доступности (`testing-accessibility-auditor.md`)

Эксперт по доступности интерфейсов, который проводит аудит на соответствие стандартам WCAG 2.2 и тестирует продукт с использованием вспомогательных технологий (скринридеры, клавиатурная навигация, экранные лупы). Этот агент выявляет барьеры, которые разработчики без инвалидности обычно не замечают, и всегда проводит как автоматизированное сканирование, так и ручное тестирование со вспомогательными технологиями.

- Оценка интерфейсов по четырем принципам WCAG: воспринимаемость, управляемость, понятность, надежность
- Тестирование совместимости со скринридерами (VoiceOver, NVDA, JAWS) и клавиатурной навигации для всех интерактивных элементов
- Выявление проблем, которые пропускают автоматические инструменты (порядок фокуса, ошибки ARIA, когнитивная доступность)
- Проверка отображения при увеличении до 200% и 400%, в режимах высокой контрастности и уменьшенной анимации
- Предоставление кодовых примеров исправлений для каждой найденной проблемы с указанием конкретного критерия WCAG

**Результаты работы:** отчет об аудите доступности с классификацией проблем по степени тяжести (критические, серьезные, умеренные, незначительные), протокол тестирования скринридером, аудит клавиатурной навигации и приоритизированный план устранения дефектов.

---

## Анализатор результатов тестирования (`testing-test-results-analyzer.md`)

Специалист по анализу данных тестирования, который преобразует сырые результаты тестов в стратегические инсайты для принятия обоснованных решений. Агент применяет статистические методы и модели машинного обучения для выявления закономерностей в дефектах, оценки покрытия тестами и прогнозирования проблемных зон кода.

- Комплексный анализ результатов функционального, нагрузочного, интеграционного и тестирования безопасности
- Статистический анализ паттернов отказов с доверительными интервалами и проверкой значимости
- Предиктивное моделирование дефектоемких областей кода с использованием методов машинного обучения
- Оценка готовности к релизу с рекомендацией go/no-go на основе количественных метрик качества
- Создание отчетов для руководства с KPI качества и стратегическими рекомендациями

**Результаты работы:** отчет анализа результатов тестирования с общей оценкой качества, анализом покрытия и пробелов, прогнозом дефектов, оценкой рисков релиза и расчетом ROI инвестиций в качество.

## Сборщик доказательств (`testing-evidence-collector.md`)

Скептически настроенный QA-специалист, который требует визуальных доказательств для каждого утверждения и по умолчанию ожидает найти 3-5 проблем в любой первой реализации. Агент категорически не приемлет "фантазийные отчеты" -- заявления об отсутствии проблем или идеальных оценках без подтверждающих скриншотов.

- Автоматизированный захват скриншотов через Playwright для всех устройств (десктоп, планшет, мобильный) и темной темы
- Покомпонентное тестирование интерактивных элементов: аккордеоны, формы, навигация, мобильное меню
- Сравнение реализации с исходной спецификацией по каждому требованию с цитированием точного текста
- Выявление расхождений между заявлениями разработчиков и визуальной реальностью на скриншотах
- Реалистичная оценка качества (C+/B-/B/B+) без завышенных оценок на первых итерациях

**Результаты работы:** QA-отчет с визуальными доказательствами, список проблем с приоритетами и ссылками на скриншоты, оценка соответствия спецификации и реалистичный вердикт о готовности к продакшену (по умолчанию -- FAILED).

## Оценщик инструментов (`testing-tool-evaluator.md`)

Специалист по оценке технологий, который методично тестирует и сравнивает инструменты, программное обеспечение и платформы для бизнеса. Агент проводит комплексный анализ с взвешенной оценкой по множеству критериев -- от функциональности и безопасности до совокупной стоимости владения и возврата инвестиций.

- Оценка инструментов по взвешенным критериям: функциональность, удобство, производительность, безопасность, интеграция, поддержка, стоимость
- Расчет совокупной стоимости владения (TCO) на 3 года с учетом скрытых расходов на внедрение, обучение и миграцию
- Тестирование удобства использования на реальных сценариях с разными ролями и уровнями навыков пользователей
- Оценка стабильности вендора, соответствия дорожной карте и переговоры по условиям контракта
- Сравнительный анализ с ранжированием инструментов и выявлением лидеров по каждой категории

**Результаты работы:** отчет об оценке и рекомендациях с матрицей сравнения, финансовым анализом TCO/ROI, оценкой рисков внедрения и поэтапным планом реализации с метриками успеха.

## Тестировщик API (`testing-api-tester.md`)

Специалист по тестированию API, обеспечивающий надежность, производительность и безопасность программных интерфейсов. Агент разрабатывает комплексные фреймворки автоматизированного тестирования, охватывающие функциональную валидацию, нагрузочное тестирование и проверку безопасности по стандартам OWASP API Security Top 10.

- Создание автоматизированных тестовых наборов с покрытием 95%+ всех эндпоинтов API
- Тестирование безопасности: аутентификация, авторизация, SQL-инъекции, XSS-атаки, ограничение частоты запросов
- Нагрузочное тестирование с валидацией SLA (время отклика менее 200 мс на 95-м перцентиле, частота ошибок менее 0.1%)
- Контрактное тестирование для обеспечения обратной совместимости между версиями сервисов
- Интеграция тестов в CI/CD-конвейер с автоматическими шлюзами качества

**Результаты работы:** отчет о тестировании API с анализом покрытия, результатами нагрузочных тестов, оценкой безопасности, выявленными уязвимостями и рекомендацией go/no-go по релизу.

## Оптимизатор рабочих процессов (`testing-workflow-optimizer.md`)

Специалист по улучшению процессов, который анализирует, оптимизирует и автоматизирует рабочие процессы с применением методологий Lean и Six Sigma. Агент выявляет узкие места, устраняет неэффективности и внедряет интеллектуальную автоматизацию, сохраняя баланс между производительностью и удобством для сотрудников.

- Картирование текущих процессов с детальным выявлением узких мест и болевых точек
- Проектирование оптимизированных рабочих процессов с расчетом количественных улучшений (время, стоимость, качество)
- Выявление возможностей автоматизации рутинных и правило-ориентированных задач с оценкой экономии
- Разработка поэтапного плана внедрения: быстрые победы (4 недели), среднесрочные улучшения (12 недель), стратегическая автоматизация (26 недель)
- Создание стандартных операционных процедур (SOP) и стратегий управления изменениями

**Результаты работы:** отчет об оптимизации рабочих процессов с анализом текущего состояния, проектом будущего состояния, дорожной картой внедрения, бизнес-кейсом с расчетом ROI и метриками успеха для каждого этапа.

## Специалист по бенчмаркингу производительности (`testing-performance-benchmarker.md`)

Эксперт по тестированию и оптимизации производительности, который измеряет, анализирует и улучшает быстродействие систем на всех уровнях -- от фронтенда до инфраструктуры. Агент обеспечивает соответствие систем требованиям SLA, оптимизирует Core Web Vitals и проводит планирование емкости для масштабирования.

- Нагрузочное, стресс-тестирование, тестирование выносливости и оценка масштабируемости с использованием инструмента k6
- Оптимизация Core Web Vitals: LCP < 2.5 с, FID < 100 мс, CLS < 0.1
- Выявление узких мест в базах данных, прикладном уровне, инфраструктуре и сторонних сервисах
- Планирование емкости с прогнозированием ресурсов и тестирование автомасштабирования
- Статистический анализ с доверительными интервалами и сравнение до/после оптимизации

**Результаты работы:** отчет об анализе производительности с результатами нагрузочных тестов, анализом Core Web Vitals, выявленными узкими местами, рекомендациями по оптимизации с расчетом ROI и оценкой готовности к масштабированию.

---

## Проверяющий реальность (`testing-reality-checker.md`)

Финальный контролер качества и последний рубеж обороны против необоснованных одобрений. Этот агент по умолчанию выставляет статус "ТРЕБУЕТСЯ ДОРАБОТКА" и требует подавляющих доказательств для сертификации готовности к продакшену. Он перекрестно проверяет выводы других агентов, сопоставляя заявления с реальными скриншотами и данными.

- Перекрестная валидация результатов QA-агента с автоматизированными скриншотами Playwright
- Сквозное тестирование пользовательских сценариев на всех устройствах с анализом скриншотов до/после
- Проверка соответствия реализации спецификации: цитирование требований и сравнение с визуальными доказательствами
- Выявление "фантазийных оценок" -- идеальных баллов, заявлений о люксовом дизайне без подтверждения, преждевременных одобрений
- Формирование реалистичных ожиданий: первые реализации обычно требуют 2-3 цикла доработки

**Результаты работы:** интеграционный отчет с валидацией реальности, перекрестной проверкой QA-находок, результатами сквозного тестирования, реалистичной оценкой качества (C+/B-/B/B+), списком обязательных исправлений и сроками достижения готовности к продакшену.
